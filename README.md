# Аппроксимация данных методом наименьших квадратов

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue)](https://www.python.org/)
[![NumPy](https://img.shields.io/badge/NumPy-1.21%2B-orange)](https://numpy.org/)
[![Matplotlib](https://img.shields.io/badge/Matplotlib-3.5%2B-yellow)](https://matplotlib.org/)

Аналитический инструмент с графическим интерфейсом для аппроксимации экспериментальных данных. Строит линейную и полиномиальные модели (2‑й и 3‑й степени), визуализирует результаты, вычисляет коэффициент детерминации \(R^2\) и даёт прогноз на три шага вперёд.

## 📋 Возможности

- **Загрузка данных** из текстовых файлов (CSV, TXT), электронных таблиц Excel (.xls, .xlsx) и ручной ввод через GUI.
- Автоматическое определение разделителей, поддержка дат в формате `YYYY-MM`.
- Построение **линейной регрессии** (\(y = ax + b\)).
- Построение **полиномиальной регрессии** 2‑й и 3‑й степени.
- Визуализация исходных точек, регрессионных кривых и прогнозных значений на одном графике.
- Расчёт коэффициента детерминации \(R^2\) для каждой модели.
- Таблица сравнения моделей с абсолютными и относительными ошибками, цветовой индикацией точности и итоговыми суммами.
- Прогноз на три шага вперёд с рекомендацией наиболее надёжной модели.
- Сохранение полного отчёта в текстовый файл.
- Крупные кнопки, полноэкранный режим, встроенная легенда цветов.

## 🛠 Требования

-   Python 3.8 или выше
- Установленные библиотеки: `numpy`, `matplotlib`, `openpyxl`, `xlrd`

## Установка зависимостей:
```bash
pip install numpy matplotlib openpyxl xlrd
```
