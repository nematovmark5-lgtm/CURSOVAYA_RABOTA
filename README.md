# Аппроксимация данных методом наименьших квадратов

 [![Python](https://img.shields.io/badge/Python-3.8%2B-blue)](https://www.python.org/)
 [![NumPy](https://img.shields.io/badge/NumPy-1.21%2B-orange)](https://numpy.org/)
 [![Matplotlib](https://img.shields.io/badge/Matplotlib-3.5%2B-yellow)](https://matplotlib.org/)

Аналитический инструмент для обработки экспериментальных данных и построения линейных и полиномиальных трендов с визуализацией результатов и оценкой качества аппроксимации (коэффициент детерминации \(R^2\)).

## 📋 Возможности

-   Загрузка экспериментальных данных из текстовых файлов (CSV, TXT) с произвольными разделителями.
-   Построение **линейной регрессии** (\(y = ax + b\)).
-   Построение **полиномиальной регрессии** 2-й и 3-й степени.
-   Визуализация исходных точек и всех полученных кривых на одном графике.
-   Расчёт коэффициента детерминации \(R^2\) для каждой модели.
-   Вывод уравнений регрессии и значений \(R^2\) в консоль.

## 🛠 Требования

-   Python 3.8 или выше
-   Установленные библиотеки: `numpy`, `matplotlib`

Установка зависимостей:
```bash
pip install numpy matplotlib
```

## 📊 Пример работы
После запуска в консоль будут выведены коэффициенты и R^2 для каждой модели, а также отобразится график.
